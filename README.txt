EXIO ROGUELIKE Ver.3 Web版

起動方法:
1. このフォルダの index.html をブラウザで開く
2. パスワード入力で「0618」を入力
3. タイトル画面で「はじめから」または「つづきから」を選ぶ

操作:
- PC: 矢印キー / Enter / Escape
- スマホ: 画面下の移動ボタン、各コマンドボタン

セーブ:
- ブラウザのlocalStorageに保存されます。
- 同じブラウザで index.html を開くと「つづきから」が使えます。

注意:
- Python/Tkinter版をHTML/JavaScriptへ移植したWeb版です。
- 素材は assets フォルダ内の画像を使用しています。
